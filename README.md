# 尘白MOD管理柒

🇨🇳 [中文](./README.md) | 🇬🇧 [English](./README_EN.md)

## 📌 快速上手

1. **设置 MOD 管理文件夹（源MOD目录）**  
   选择一个用于存放 MOD 文件的文件夹。建议与 MOD 管理器所在的目录保持一致。

2. **设置游戏 MOD 安装目录（目标MOD目录）**  
   将此路径指向游戏实际的 MOD 存放位置。默认路径示例：
   `游戏文件夹/Game/Content/Paks/mods`

3. **添加你的 MOD**  
   解压下载好的 MOD 压缩包，然后将文件复制到第 1 步设置的 MOD 管理文件夹中，可按你的习惯分类整理。

   > 📸 **预览图**：如果你在与 `.pak` 文件相同的文件夹中放入一个同名的图片文件（png、jpg、bmp 或 webp），它将被自动识别为该 MOD 的预览图。

4. **刷新并开始管理**  
   点击管理器上的"刷新"按钮。就这样 —— 你现在可以随时启用、禁用并整理你的 MOD 了。

## ⚙️ 操作指南

### 源MOD目录
你的 MOD 管理文件夹，用于存放所有 MOD 文件（建议自行分类整理）。可点击"选择..."选择文件夹，或点击"打开"在资源管理器中打开。

### 目标MOD目录
游戏读取 MOD 的目录，通常为 `游戏文件夹/Game/Content/Paks/mods`。如果没有 `mods` 目录，需要手动创建。可点击"选择..."设置路径，或点击"打开"快速定位目录。

### 清空目标目录
点击"清空目标目录"可从游戏 MOD 安装目录中移除所有已安装的 MOD。

### 免打扰模式
开启后，会隐藏不包含 `.pak` 的无关项，让列表更清爽。

### 排序
支持按名称/时间的升序与降序排序，并会尽量保留你当前展开的目录状态。

### 全部展开/全部收起
若当前选中了某个目录，将只对该目录递归展开/收起；否则对整个列表展开/收起。

### 新建文件夹
在左侧选中一个目录后点击"新建文件夹"，会在该目录下创建子目录；若选中的是文件，则在其同级创建；未选中任何项则默认在源MOD目录下创建。

### 启用/禁用
勾选 `.pak` 文件后点击"启用/禁用"。支持勾选文件夹进行批量选择；也可双击 `.pak` 文件快速切换，或在列表上右键使用菜单快捷启用/禁用。

### 刷新
当你新增/移动/删除 MOD 文件后，点击"刷新"重新读取源目录结构，并更新启用状态与预览信息。

### 记忆启用状态
底部按钮栏可"记忆当前启用状态"保存一套启用/禁用配置；之后可从下拉框选择并"应用记忆"，也可"删除记忆"。

### 预览图操作
右侧预览区支持拖拽本地图片、从网页拖拽图片/链接（自动下载并关联），也支持直接粘贴剪贴板图片。双击右侧预览图可用系统图片查看器打开；在左侧列表对图片文件右键可删除。

### 粘贴（Ctrl+V）
在左侧选中目标目录（或任意文件），按 `Ctrl+V` 可将剪贴板中的文件/文件夹粘贴到该目录；支持资源管理器复制，也支持从系统 zip / WinRAR 复制（保留目录结构）。

### 打开文件
双击文本/图片文件会调用系统默认程序打开查看内容。

### 表格调整
可拖动列分隔线调整列宽，列宽会自动记忆。

### 语言
右下角可切换界面语言。
